# Semantic Release Config 

[![version][npm-image]][npm-url]

> [`semantic-release`][semantic-release] shareable config

## Usage

###### `.releaserc`

```bash
{
  "extends": "@telus/semantic-release-config"
}
```

```bash
npm install @telus/semantic-release-config
npx semantic-release
```
